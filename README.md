Emoji Memory Match
A fun, browser-based memory game built with JavaScript. Challenge your brain by matching pairs of emojis before the timer runs out!

Features:

Dynamic Gameplay: Cards are shuffled randomly every time you reset.
Performance Tracking: Real-time counter for Moves and Time.
Celebration Effects: Custom fireworks animation and a congratulatory modal when you win.
Responsive Design: Playable on both desktop and mobile browsers.

Built With:

HTML5 - Semantic structure.
CSS - Flexbox layout and keyframe animations for fireworks.
JavaScript - Logic for card flipping, matching, and game state.
